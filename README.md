# Backend-12
### Описание
Яндекс Практикум, 12 спринт. Серверная разработка на Node.js, Express.js.
### Развертывание
Развертывание с помощью скрипта **start**. 
```bash
npm run start
```
Для отладки и внесения изменений используется режим **dev** с "hot reload".
```bash
npm run dev
```
